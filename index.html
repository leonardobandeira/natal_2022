Desde 2014...

em uma ação que começou na família para agradecer por uma graça recebida, a Washimara faz doações de brinquedos para crianças carentes no Natal. Aos poucos, com ajuda de amigos, o projeto foi crescendo e eles conseguem atender cada vez mais famílias. Ela já entregou presentes até debaixo de chuva, vestida de Papai Noel, para dar um pouco mais de alegria para as crianças nessa data tão especial. Por isso, escolhemos o projeto dela como nossa “Estrela de Natal de 2022”
